# 🚀 CodSoft Internship - Authentication UI

## 📌 Project Overview

This project is a modern and responsive **Login & Signup UI** built as part of the CodSoft Internship.
It focuses on clean design, smooth user experience, and mobile-first responsiveness.

---

## ✨ Features

* 🔐 Login & Signup Interface
* 📱 Fully Responsive (Mobile Friendly)
* 👁️ Show/Hide Password Toggle
* 🎨 Modern Gradient UI Design
* 🧊 Glassmorphism Effects
* ⚡ Smooth UI Transitions
* 🧠 Basic Form Validation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
